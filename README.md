# GitMemo

## Development

### 🔌　Setup

pnpm を利用して依存関係をインストールする。

```sh
pnpm install
```

開発環境を立ち上げる。

```sh
pnpm run dev
```

### 📦 Create and run npm package locally

ローカル環境で npm パッケージを作成して `playground` ディレクトリで実行する。

```sh
pnpm run pack
```

これは、`npx gitmemo` と同じ挙動になると思っており、Vite で serve されている。
